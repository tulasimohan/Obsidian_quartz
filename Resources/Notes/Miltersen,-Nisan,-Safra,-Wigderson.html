Introduction

In a static data structure problem, we are given a domain D of possible data, a domain Q of possible queries, and a map f:Q×D→A, where f(x,y) is the answer to query x about data y.

Static Data Structure Problem (Static DSP)

A Static Data Structure Problem is given by the tuple (D,Q,A,f) where

  • D is the domain of possible data,
  • Q is the domain of possible queries,
  • A is the set of possible answers and
  • f:Q×D→A is a map where f(x,y) is the answer to query x about data y.

Typically the answer set A={0,1}, unless specified explicitly the domain is assumed to be boolean values. We usually drop A from the specification of the Static DSP.

A solution with parameters s, b, and t is a method of storing any y∈D as a data structure ϕ(y) in the memory of a random access machine, using s memory cells, each containing w bits, so that any query in Q can be answered by accessing at most t memory cells.

Solution/ Scheme (s,w,t)

A (s,w,t) Solution to a Static Data Structure Problem (D,Q,f) is given by the tuple (s,t,ϕ,{Qi​}) where

  • ϕ:D→[2w]s is an encoding map
  • Qi​ is a decision tree of depth at most t for each query i∈Q.

Communication Complexity Prelims

[a,b]-protocol

An [a,b]-protocol for f is a protocol where the total number of bits that Alice sends Bob is at most a and the total number of bits that Bob sends Alice is at most b.

[t,a,b]A-protocol

A [t,a,b]A-protocol for f is a protocol where each of Alice’s messages contains at most a bits and each of Bob’s messages contains at most b bits and at most t messages are sent, with Alice sending the first message. A [t,a,b]B protocol is defined similarly.

Communication matrix

Identify f with the matrix M with Mx,y​=f(x,y); i.e., index the rows by Alice’s possible inputs, and the columns by Bob’s possible inputs.

Richness Technique

(u,v)-richness

We say that a matrix (and a problem) is (u,v)-rich if at least v columns contain at least u 1-entries.

Lemma 5 Richness lemma

Let f be a (u,v)-rich problem. If f has a randomized one-sided error [a,b]-protocol, then f contains a submatrix of dimensions at least u/2a+2×v/2a+b+2 containing only 1-entries.
