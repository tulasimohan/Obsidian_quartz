# Range Query


>[!Note] Range Query $RQ_{n,\ell}$
> **Alice** gets an interval $[x_1 , x_2]$, 
> **Bob** gets a set $y \subseteq U$ of size at most $\ell$, and 
> the players must agree on an element in $[x_1 , x_2] \cap y$ if one exists.



- $RQ_{n,\ell}$ has a $[O(1), \log n + O(\log \ell), n]$-protocol. (Theorem 17, [[On Data Structures and Asymmetric Communication Complexity\|On Data Structures and Asymmetric Communication Complexity]])
