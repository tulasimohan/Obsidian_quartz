# Dictionary

>[!Note] Dictionary Problem #static_DSP  
>- $D$ is the set of subsets $y\subseteq \{0, \dots , N-1\}$ of a certain size, 
>- $Q$ is the set $[0, ..., N-1]$, and
>- $f (x, y)=1$ if and only if $x \neq y$.
^89c656

Also known as Membership. 

Communication complexity version is [[../../Problems/Communication_Complexity/Membership\|Membership]] 


Andersson et al showed that Static dictionary $t = \Theta(\sqrt{ \frac{\log n}{\log \log n}  } )$
