
Consider the following collection of segments in the plane: 
$\{ [ jd^i, (j + 1)d^i ] \times i | 0 \leq i \leq h, 0 \leq j < d^{h−i} \}$, corresponding to a regular $d$-ary tree $T$ of height $h$. 

When a node of T gets marked or unmarked, the corresponding segment is inserted or removed from the collection.
