
>[!DSP] Pagh’s problem. 
>Motivated by information retrieval applications that employ set intersection queries on inverted lists, this problem asks to 
>**Data:** preprocess a family of sets $X_1, X_2, \dots \subseteq [n]$ and support:  
>- **updates:** create a new set $X_{new} = X_i \cap X_j$, where $X_i$ and $X_j$ remain intact; 
>- **queries:** whether some element $z$ belongs to some set $X_i$.
