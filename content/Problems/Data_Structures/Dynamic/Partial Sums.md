# Partial Sums

>[!DSP] Partial Sum mod $k$. 
>**Data:** An array A[1],…, A[N] of integers mod $k$ is to be represented. 
>**Updates:** are 
>*add(i, δ):* which implements $A[i] ← A[i] + δ$; and 
>**Queries:** are 
>*sum(j):* which returns $\sum_{i≤j}A[i] (\mod k)$.

This problem is demoted $PS(n, k)$.
