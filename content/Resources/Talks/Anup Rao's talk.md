### Log Rank Conjecture


>[!conj] Log-rank conjecture [Lovasz-Saks]
$\exists c >1$ such that
$$CC(f) \leq \log^c rank(M_f)$$

rank ($rk$):  
non-neg rank ($rk_+)$: min $r$ such that $M_f$ can be expressed as sum of rank 1 matrices.  


[Lovett], building on [Lovett, Gavinsky]
- showed that $CC(F) \leq \sqrt{rk(f)} \cdot \log rk(f).$
- Uses theorem from convex geometry called John's theorem
- ***Key lemma:*** If a boolean matririx has rank $r$, it contains a monochromatic rectangle of density $\frac{1}{\sqrt{r}}$. 

[Lovasz]
- showed $\log (rk_+(f)) \leq CC(f) \leq \log^2 (rk_+(f))$. 
- For non-boolean matrices, $rk(f)$  and $rk_+(f)$ can be exponentially related.
